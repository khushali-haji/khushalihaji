Where Financial Vulnerability Lives

NFCS State-by-State Tracking · 2009–2024 · weighted to each state

Each state is a 3D block: color = financial vulnerability (darker = more vulnerable — averaging five hardships: struggling to pay bills, no 3-month emergency fund, spending more than income, overdrawing checking, predatory borrowing) and height = average household income (taller = higher). So a tall dark block = high-income but high-vulnerability; a short dark block = the poor-and-struggling pattern. Drag to rotate, scroll to zoom, hover for the breakdown, use the year timeline, and filter by income band.
